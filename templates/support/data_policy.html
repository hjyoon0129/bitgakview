1. 데이터의 성격

BitgakView의 차트, 시세, 거래량, 지표, 시장지표, 검색 결과는 투자 참고용 정보입니다. 데이터는 외부 데이터 제공처, 공개 데이터, 자체 계산 로직, 캐시를 통해 표시될 수 있습니다.

2. 지연·오류·누락 가능성

네트워크, 데이터 제공처, 서버 상태, 장 마감 처리, 휴장일, 종목 변경, 액면분할, 상장폐지, API 제한 등에 따라 데이터가 지연되거나 오류·누락될 수 있습니다.

3. 실시간 거래 사용 제한

BitgakView는 증권사 주문 시스템이 아니며, 실시간 주문 체결 또는 매매 결정을 위한 공식 시세 단말이 아닙니다. 실제 거래 전에는 증권사 또는 거래소에서 제공하는 최신 정보를 확인해야 합니다.

4. 재배포 제한

이용자는 BitgakView에서 제공되는 시세, 차트, 지표, 검색 데이터, 화면 구성 요소를 운영자의 허가 없이 대량 수집, 복제, 저장, 재판매, API화, 재배포할 수 없습니다.

5. 정정 및 문의

데이터 오류를 발견한 경우 고객센터를 통해 종목명, 코드, 발생 시각, 화면 캡처, 문제 내용을 함께 보내주시면 확인하겠습니다.
